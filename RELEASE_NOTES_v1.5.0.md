# CareerAgent v1.5.0 Release Notes

## 新增：知识库问答

本版将现有的采集、文本化、入库、Embedding、混合检索与评测链路连接为可直接演示的 RAG 问答闭环。

### 主要能力

- 独立“知识库问答”页面；
- RRF / 加权混合 / 纯向量三种检索模式；
- 多来源自动扩大 Top K 与 MMR 多样性；
- 低置信度单文档问题自适应 Reranker；
- 非思考模式的一次聊天模型回答；
- 严格编号引用与真实视频链接；
- 相邻片段补充、上下文字符预算和输出 Token 预算；
- 资料不足与无有效引用时安全回退。

### 默认设置

```text
检索：RRF 混合，k=60
单文档 Top K：5
多来源 Top K：10
MMR：开启，lambda=0.75
Reranker：仅低置信度单文档问题
回答来源：单文档最多5篇，多来源最多8篇
上下文：最多12000字符
回答：最多1200 tokens，关闭思考模式
```

### 升级说明

不需要重新建立已有索引，也不会修改数据库、人工最终稿、评测题或 API Key。覆盖 `app` 文件夹并重启即可。
