CareerAgent v1.23.0 使用说明
===========================

首次运行：
1. 安装 Python 3.11 或 3.12。
2. 安装并启动 Docker Desktop。
3. 双击 CareerAgent_Start.bat。
4. 选择运行数据目录和导出目录。
5. 等待 PostgreSQL、依赖和数据库迁移检查完成。
6. 浏览器自动打开后，首次采集前完成抖音登录。

多层情报问答：
- 普通 RAG：查原文、参数和单篇内容。
- 行业情报分析：查趋势、观点、事件、统计和博主比较。
- 自动判断：系统先识别问题意图，再选择证据层。

注意：
- 代码仓库不包含 Cookie、API Key、数据库、模型或用户导出数据。
- 长任务可在后台任务中心查看进度、取消或重试。
- 关闭黑色启动窗口会停止本地服务。
- PostgreSQL 恢复操作会覆盖数据，必须先备份并人工确认。
