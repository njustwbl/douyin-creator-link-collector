CareerAgent v1.9.3 使用说明
==========================

首次使用
--------
1. 完整解压项目。
2. 双击 CareerAgent_Start.bat。
3. 首次启动选择运行数据目录和导出目录。
4. 等待程序自动创建虚拟环境、安装基础依赖和 Chromium。
5. 浏览器打开后，先在页面中完成抖音登录。
6. ASR、Ollama、Embedding 和 Reranker 模型按需下载，不包含在代码包中。

开发者启动
----------
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
playwright install chromium
uvicorn app.main:app --reload

本地数据
--------
数据库、浏览器登录状态、API Key、模型、缓存、日志和导出文件均保存在本地配置目录，不应上传 GitHub。

停止程序
--------
关闭黑色运行窗口即可停止本地服务。
