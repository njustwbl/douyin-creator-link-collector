# CareerAgent v1.6.0 发布说明

## 新增：评测与优化闭环

本版在现有“检索评测”和“知识库问答”之间补齐了反馈闭环：

1. 将检索评测题导入端到端 RAG 评测集；
2. 为题目补充参考答案、关键要点、拒答预期和调参集/测试集标签；
3. 运行真实检索与回答链路；
4. 保存检索结果、上下文、答案、引用、耗时、请求和 Token；
5. 自动诊断失败发生在召回、排序、上下文选择、回答、引用、拒答还是工程成本层；
6. 用同一套检索评测题批量比较参数方案；
7. 将综合最优方案设为正式知识库问答默认配置。

## 成本控制

- 每道端到端评测题只执行一次正常问答流程；
- 规则评测不调用额外裁判大模型；
- 参数实验只跑检索评测，不调用回答模型；
- 支持限制端到端评测题数和并发；
- Reranker 继续按低置信度策略按需触发。

## 使用建议

- 调参集：约 70% 题目，用于选择模型和参数；
- 测试集：约 30% 题目，只在参数确定后验证一次；
- 关键要点建议每行一个，用于低成本评估答案覆盖度；
- “答案正确性代理”和“忠实度代理”属于规则指标，适合回归比较，不等同于人工语义评分。
