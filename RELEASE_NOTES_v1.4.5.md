# CareerAgent v1.4.5 Release Notes

## 主要修复

- 基于完整项目重做“单题检索”和“批量评测”的参数排版。
- 参数按“基础检索 / 多来源召回 / 精排策略”分组，无效参数自动隐藏。
- 首页、CSS、JavaScript 全部使用 no-store，并由后端动态注入当前版本号。
- 启动器识别旧 CareerAgent 进程；默认端口被旧版本占用时自动使用 8001～8020。
- 页面右上角同时显示界面版本、后端版本和 PID；版本不一致时显示项目路径告警。
- `/health` 增加项目根目录、Web 目录和资源签名，便于定位启动了错误副本的问题。
- 修正 Whisper CPU 回退测试的句末标点预期，完整测试 67 项通过。

## 启动检查

启动后右上角应显示：

```text
本地服务正常
界面 v1.4.5 · 后端 v1.4.5 · PID ...
```

如果版本不一致，页面会直接显示旧服务所在的项目路径。
