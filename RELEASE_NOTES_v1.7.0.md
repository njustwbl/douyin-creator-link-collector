# CareerAgent v1.7.0 发布说明

## 目标

在保留现有 API 能力的前提下，新增一套可独立测试的本地 Qwen 路径。当前版本不做自动模型路由，也不把清洗、入库、建索引和问答强制串成自动流水线。

## 主要变化

### 文本清洗与纠错

- 删除旧的本地 1.5B 纠错模型运行路径；
- “本地”模式改用 Ollama `qwen3.5:4b`；
- 基础清洗、术语纠错、逐段安全校验和局部回退继续保留；
- 本地调用固定串行、关闭思考输出并动态限制输出长度。

### RAG 知识入库准备

- 增加“本地 Qwen3.5-4B / API 大模型”手动选择；
- 本地模式使用结构化 JSON 输出并校验字段；
- 同一正文可分别生成本地版和 API 版结果，provider/model 会写入记录。

### 知识库检索

- 新增“Ollama 本地向量”提供商；
- 默认本地模型为 `qwen3-embedding:4b`；
- 使用 Ollama 原生 `/api/embed`，无需 API Key；
- 本地和 API Embedding 分别建立独立索引。

### 知识库问答

- 增加“本地 Qwen3.5-4B / API 大模型”手动选择；
- 本地模式继续强制 JSON、编号引用和证据不足判断；
- 默认推荐 3/5 个来源、8000 字符上下文和 800 Token 输出。

### Ollama 管理

新增统一设置与测试接口：

- Base URL；
- 对话模型；
- Embedding 模型；
- 上下文长度；
- 超时；
- 模型保活时间；
- Embedding 批量大小；
- 服务状态、对话模型和 Embedding 模型连接测试。

## 安装模型

```powershell
ollama pull qwen3.5:4b
ollama pull qwen3-embedding:4b
```

## 兼容性

- 数据库结构无需手工迁移；
- 旧索引继续保留；
- 旧 1.5B 权重文件不会自动删除，但代码不再调用；
- API 模式和现有评测流程继续可用。
